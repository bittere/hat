# Security Policy

## Supported Versions

Only the latest major version eg. `v0.<anything>.<anything>` on GitHub Releases is supported for now. As Hat matures, older major versions will receive security updates.

## Reporting a Vulnerability

For any vulnerability in **how a particular library/dependency works**, file a report in the respective dependency eg. if `tauri` has a security flaw, go to their repository and report it there.
